Array Types
An array can be of these types:

```
Types		Example Definition
--------------- --------------------
Number		[]int,[]i64
String		[]string
Rune		[]rune
Boolean		[]bool
Array		[][]int
Struct		[]MyStructName
Channel		[]chan f64
Function	[]MyFunctionType []fn (int) bool
Interface	[]MyInterfaceName
Sum Type	[]MySumTypeName
Generic Type	[]T
Map		[]map[string]f64
Enum		[]MyEnumType
Alias		[]MyAliasTypeName
Thread		[]thread int
Reference	[]&f64
Shared		[]shared MyStructType

```
